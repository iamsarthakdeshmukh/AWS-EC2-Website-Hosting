# Website Hosting on AWS EC2

## 📌 Project Overview

This project demonstrates how to host a basic static website on an AWS EC2 instance using Nginx.
The server runs on Ubuntu Linux and the website is accessible through the public IP address of the EC2 instance.

This project provides hands-on experience in cloud computing, Linux server setup, and basic networking.

---

## 🚀 Tech Stack

* AWS EC2
* Ubuntu Server
* Nginx Web Server
* Git
* Linux

Technologies used:

* Amazon Web Services – Cloud platform used to launch the server
* Ubuntu – Operating system running on EC2
* Nginx – Web server used to host the website
* Git – Version control system for managing project files

---

## ⚙️ Project Workflow

1. Created an EC2 instance on AWS.
2. Selected Ubuntu Server as the operating system.
3. Configured security groups to allow SSH and HTTP traffic.
4. Connected to the EC2 instance using SSH.
5. Installed and configured Nginx web server.
6. Created a simple HTML website.
7. Hosted the website using Nginx.
8. Accessed the website using the public IP of the EC2 instance.
